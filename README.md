# New React Project Boilerplate

An alternative to using create-react-app with Parcel

## How to use

### Clone

```bash
npm install -g degit


degit https://github.com/DarrenXu94/new-react-app#main
```

Degit removes all the repository git information.

### Install

```bash
npm i
```

### Run

```bash
npm start
```
